# CyberMentor

Интеллектуальный Telegram-агент для сопровождения профессионального самоопределения школьников, абитуриентов и студентов в области кибербезопасности.

## Возможности

- Профориентационное тестирование
- Определение направления Blue Team / Red Team
- Подбор профессий
- Персональный карьерный план
- ИИ-наставник (OpenAI)
- Рекомендации по развитию
- Профиль пользователя

## Стек технологий

- Python 3.12
- Aiogram 3
- OpenAI API
- JSON (в перспективе SQLite)
- Docker

## Запуск

```bash
pip install -r requirements.txt
python main.py
```
